## Creator Commands
Commands for the master user `!<command>`

| Command                    | Description                               |
|:---------------------------|:------------------------------------------|
| `adminstats`               |Displays bot adminstrative data            |
| `mkdir`                    |Sets status message to input string        |
| `rmdir`                    |Displays twitter channel ID                |
| `writeFile`                |Displays twitch channel ID                 |
| `deleteFile`               |Displays youtube channel ID                |
